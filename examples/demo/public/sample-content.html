Welcome to Washi UI

A powerful HTML commenting library for modern web applications. Click anywhere to add annotations!

Getting Started

Washi UI makes it easy to add collaborative commenting to any HTML content. Simply mount the library to an iframe, and your users can start adding annotations immediately.

Quick Setup

Install the package, create an adapter for your storage backend, and you're ready to go. The library handles all the complexity of positioning, scrolling, and event management.

Key Features

Pin-Based Comments

Users can click anywhere on the content to place comment pins at precise locations.

Scroll Sync

Comment pins stay perfectly aligned with content even as users scroll.

Framework Agnostic

The core library works with any framework. Use our React bindings or build your own.

Adapter Pattern

Bring your own storage backend - REST API, GraphQL, Firebase, or local storage.
